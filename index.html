Load Balancing Reverse Proxy Server

Course Project• October 2017 - November 2017

This project was a part of the course Network Programming. A simple load balancing server was written in Python with the asyncio library, which distributed load in a Round Robin manner. For testing, the backend servers were modelled by CPU cores and the service requested was a block mining operation (Hash Generation) with a set difficulty.

Analysing the performance of OpenACC for simple algorithms and learning GPU Architecture

Semester Project• September 2017 - November 2017

The goal of the project was to analyse OpenACC performance and learn about the GPU to parallelise Random Forest Generation efficiently. Performance boosts of upto 37% were observed for tasks like Matrix Multiplication, Fast Fourier Transform as the used the GPU.
